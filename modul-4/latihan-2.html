FAQ

Apa itu CSS Grid?

CSS Grid adalah sistem layout berbasis grid untuk membuat tampilan web.

Apa itu Flexbox?

Flexbox digunakan untuk layout satu dimensi (baris atau kolom).

Apa itu pseudo-element?

Pseudo-element digunakan untuk styling bagian tertentu dari elemen.
